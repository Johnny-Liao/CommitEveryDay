# Hello GDB

1. g++ -g -Wall name.cpp -o name 

2. gdb -tui name 
```
b : set break with line on condition or on function --- eg : b 11 if var = xxx
r : run
n : next
s : step
bt: backtrace
info locals : show the information of locals
p : print value of var
pt: print type of var
set var name = xxx : YKI
delete
continue
q: quit
```
