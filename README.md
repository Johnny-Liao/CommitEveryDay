# CommitEveryDay
Make a little progress every day.

# 目标
每天提交一些有意义的测试代码，养成每天编码的好习惯。但是拒接提交无意义的代码，拒绝为提交代码而提交代码。

## 用代码代替博客
是使用代码加注释这种简单直观的方法，代替一些生涩的概念性文字说明。但博客还是更锻炼人的抽象和总结能力的，有时间和精力的情况下还是尽力写博客来提升自己。

## 每日笔记
notes.doc 中为每天的学习笔记，旨在总结每天的学习内容，方便复习和提高自我总结能力。

## 近期总结
经过前一阵子的强化训练编码能力和算法实现能力有所提升，开始准备面试内容，也好久没每天更新此项目代码了，so embarrasse!
保留项目名以纪念前一段时间的强化训练，以后不定时更新。
